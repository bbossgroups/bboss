Subject: SvnAnt @SVNANTVERSION@ Available

After much anticipation, the SvnAnt @SVNANTVERSION@ is available for
download!  @SVNANTVERSION@ is both a feature release, and extensive bug fix
release.

SvnAnt provides a set of Ant <http://ant.apache.org/> task and data
types supporting non-interactive build or continuous integration
automtion.


--------------------------  IMPORTANT NOTE  --------------------------
This release of SvnAnt is built against Subversion 1.5 (JavaHL or
CLI), and has not been tested with previous Subversion releases.  JavaHL is not
included, and must be installed separately to be used in lieu of the
CLI.
